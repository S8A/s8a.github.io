import numpy as np
import math as m
import matplotlib.pyplot as plt

def euler(f, x0, y0, xf, n):
    """
    Produce la solucin aproximada mediante el mtodo 
    de Euler de la EDO y' = f(x, y) con la condicin 
    inicial y(x0) = y0, dividiendo el intervalo 
    (x0, xf) en n subintervalos uniformes.
    """
    x    = np.zeros(n)
    y    = np.zeros(n)
    dx   = (xf-x0)/n
    x[0] = x0
    y[0] = y0
    for i in range(n-1):
        x[i+1] = x[i] + dx
        y[i+1] = y[i] + dx * f(x[i], y[i])
    return x, y

def rk4(f, x0, y0, xf, n):
    """
    Produce la solucin aproximada mediante el mtodo 
    de Runge-Kutta de orden 4 de la EDO y' = f(x, y) 
    con la condicin inicial y(x0) = y0, dividiendo el 
    intervalo (x0, xf) en n subintervalos uniformes.
    """
    x    = np.zeros(n)
    y    = np.zeros(n)
    dx   = (xf-x0)/n
    x[0] = x0
    y[0] = y0
    for i in range(n-1):
        x[i+1] = x[i] + dx
        s1 = f(x[i], y[i])
        s2 = f(x[i] + 0.5*dx, y[i] + 0.5*dx*s1)
        s3 = f(x[i] + 0.5*dx, y[i] + 0.5*dx*s2)
        s4 = f(x[i] + dx, y[i] + dx*s3)
        y[i+1] = y[i] + (dx/6.0)*(s1 + 2*s2 + 2*s3 + s4)
    return x, y

Ejercicio 1¶

Se sabe que la solucin exacta al problema de valor inicial: $$ \left\{ \begin{align*} y' &= \sin(y) \\ y(a) &= y_a \end{align*} \right. $$

est dada por: $$ y(t) = 2\arctan\left(e^{t-a} \tan\left(\frac{y_a}{2}\right)\right) + 2\pi\left\lfloor\frac{y_a + \pi}{2\pi}\right\rfloor $$

donde el smbolo $\lfloor...\rfloor$ denota la funcin parte entera (piso). Calcule la solucin aproximada por Euler del anterior problema de valor inicial:

  • a) tomando $a = 0$, $y_a = 0$
  • b) tomando $a = 0$, $y_a = 100$

En cada caso utilice valores de $\Delta x$ de la forma $0.1\times2^{-k}$ donde $k\in\{0, 1, 2, 3, 4, 5\}$. Realice un grfico donde compare todas las soluciones numricas de una misma condicin inicial (para cada condicin inicial).

1.a.¶

Tenemos que resolver $y' = \sin(y)$ con la condicin inicial $y(0) = 0$. Primero lo intentaremos de forma analtica.

Reescribimos la ecuacin y separamos: $$ y' = \sin(y) \\ \frac{dy}{dt} = \sin(y) \\ \frac{dy}{\sin(y)} = dt $$

Integrando ambos lados de la ecuacin: $$ \int\!\frac{1}{\sin(y)}\,dy = \int\!dt \\ \int\!\csc(y)\,dy = t \\ \ln\left(csc(y) - \cot(y)\right) = t $$

No resulta aparente una manera de despejar $y$ en trminos de $t$, por lo que se demuestra la necesidad de una solucin numrica.

Para la solucin exacta nos proveen la funcin $y(t)$: $$ y(t) = 2\arctan\left(e^{t-a} \tan\left(\frac{y_a}{2}\right)\right) + 2\pi\left\lfloor\frac{y_a + \pi}{2\pi}\right\rfloor $$

Implementada en Python sera:

def y1(t, a, ya):
    """Funcin y(t) generalizada"""
    return (2 * np.arctan(np.exp(t - a) * np.tan(ya / 2.0)) 
           + 2 * np.pi * np.floor((ya + np.pi) / (2 * np.pi)))

def y1a(t):
    """Funcin y(t) con la condicin y(0) = 0"""
    return y1(t, 0, 0)

Tenemos que hallar la solucin numrica mediante el mtodo de Euler, por lo que necesitamos definir una funcin $f(t, y)$ como el lado derecho de $y'$, un intervalo $(x_0, x_f)$ donde se encuentre la solucin, y el nmero divisiones en las que se partir el intervalo (n).

Vamos a usar el intervalo $(0, 1)$. En lugar del nmero de divisiones, el enunciado nos pide que utilicemos valores $\Delta x = 0.1 \times 2^{-k}$, donde $k\in\{0,1,2,3,4,5\}$. A partir de $\Delta x$ obtenemos el nmero de divisiones para cada caso: $$ \Delta x = \frac{x_f - x_0}{n} \Rightarrow n = \frac{x_f - x_0}{\Delta x} $$

A continuacin se implementa esta informacin y se aplica el mtodo de Euler en Python para graficar la solucin exacta de la funcin y sus aproximaciones para cada valor de $\Delta x$:

def graficar1(sol, xmin, xmax, approx, f, x0, y0, xf, 
             divs, fmts, lbls, ejes=False):
    """Funcin general para graficar PVI con un metodo
    determinado de aproximacin numrica.
    """
    x = np.linspace(xmin, xmax)
    y = sol(x)
    fig, ax = plt.subplots()
    if ejes:
        ax.axhline(0.0, color='k')
        ax.axvline(0.0, color='k')
    for i, d in enumerate(divs):
        xx, yy = approx(f, x0, y0, xf, d)
        plt.plot(xx, yy, fmts[i], label=lbls[i])
    ax.plot(x, y, 'k', label='Solucin exacta.')
    ax.set_xlabel('x')
    ax.set_ylabel('y')
    plt.legend()
    plt.show()

def f1(x, y):
    """Lado derecho de y' = sen(y)"""
    return np.sin(y)

def nd(x0, xf, dx):
    """Nmero de divisiones dado el intervalo y dx"""
    return int((xf - x0)/dx)

def graficar_1a(xf):
    x0 = 0
    y0 = 0
    dxs = [0.1*2**(-k) for k in range(0, 6)]
    divs = [nd(x0, xf, dx) for dx in dxs]
    fmts = ['r-*','b-v','g-^','y-*','m-v','c-^']
    lbls = [r'$\Delta x$ = ' + str(dx) for dx in dxs]
    graficar1(y1a, x0, xf, euler, f1, x0, y0, xf, 
              divs, fmts, lbls)

Ejercicio 2: Euler vs RK4¶

Considere los siguientes PVI:

  • $y' = 3x(y+4)^2$, $y(0)=0$
  • $y' = e^{x+y}$, $y(0)=0$

Para cada problema de valor inicial:

  • Obtenga la solucin analtica usando el mtodo de separacin de variables.
  • Utilice los mtodos de Euler y RK4 para obtener las soluciones numricas correspondientes usando $\Delta x = 0.1$
  • Grfique las soluciones numericas junto con la exacta y compare cul mtodo funciona mejor en cada caso.

2.a.¶

Tenemos que resolver el siguiente problema de valor inicial:

$$ \left\{ \begin{align*} y' &= 3x(y+4)^2 \\ y(0) &= 0 \end{align*} \right. $$

Primero resolvemos la ecuacin diferencial de forma analtica por separacin de variables:

$$ \begin{align*} y' &= 3x(y+4)^2 \\ \frac{dy}{dx} &= 3x(y+4)^2 \\ \frac{dy}{(y+4)^2} &= 3x\,dx \\ \int\!\frac{1}{(y+4)^2}\,dy &= \int\!3x\,dx \end{align*} $$

Para la integral de la izquierda hacemos un cambio de variable:

$$ u = y + 4 \Rightarrow du = dy \\ \\ \begin{align*} \int\!\frac{1}{u^2}\,du &= \int\!3x\,dx \\ \int\!u^{-2}\,du &= \int\!3x\,dx \end{align*} $$

Integrando a ambos lados tenemos la solucin general:

$$ \begin{align*} \frac{u^{-1}}{-1} &= 3\cdot\frac{x^2}{2} + C \\ -\frac{1}{u} &= \frac{3}{2}x^2 + C \\ -\frac{1}{y + 4} &= \frac{3}{2}x^2 + C \\ \end{align*} $$

Calculamos la constante de integracin dada la condicin inicial $y(0) = 0$:

$$ -\frac{1}{0 + 4} = \frac{3}{2}(0)^2 + C \\ \Rightarrow C = -\frac{1}{4} $$

Luego reemplazamos esta constante en la solucin general y despejamos la $y$ en trminos de $x$ para tener la solucin particular de la ecuacin diferencial:

$$ \begin{align*} -\frac{1}{y + 4} &= \frac{3}{2}x^2 - \frac{1}{4} \\ \frac{1}{y + 4} &= \frac{1}{4} - \frac{3}{2}x^2 \\ \frac{1}{y + 4} &= \frac{1 - 6x^2}{4} \\ 4 &= (y+4)(1 - 6x^2) \\ \frac{4}{1 - 6x^2} &= y + 4 \\ y &= \frac{4}{1 - 6x^2} - 4 \\ \end{align*} $$

Antes de continuar, como la funcin es racional, hay que hallar las asntotas en el eje horizontal para evitarlas. Estas corresponden a los valores para los cuales el denominador es nulo:

$$ 1 - 6x^2 = 0 \Rightarrow x = \pm\sqrt{\frac{1}{6}} \Rightarrow x \approx \pm0,408248290463863 $$

Una vez obtenida esta solucin particular exacta del problema, procedemos a obtener la solucin numrica por el mtodo de Euler y su error porcentual en cada punto. Nos piden que utilicemos un valor de $\Delta x = 0.1$, por lo que hay que ajustar el nmero de divisiones (n) de acuerdo a esto. A continuacin se presenta la implementacin en Python de estos procesos:

import pandas as pd

def f2a(x, y):
    """Lado derecho de y' = f(x, y)"""
    return 3*x * (y + 4)**2

def sol2a(x):
    """Solucin particular del problema"""
    return 4/(1 - 6*x**2) - 4

def error_porcentual(x, y, f):
    """Error porcentual del valor aproximado con respecto
    al valor exacto de la funcin dado por f(x).
    """
    fx = f(x)
    if fx == 0: return 0.0
    return 100.0 * np.absolute((y - f(x)) / f(x))

def generar_tabla(f, x, y, error):
    """Genera una tabla Pandas con los valores aproximados
    (x, y), el valor exacto f(x) y el error relativo a este.
    """
    return pd.DataFrame({
        'x': x,
        'y': y,
        'y(x)': [f(xi) for xi in x],
        'Error relativo (%)': error
    })

def resolver_2a(metodo):
    """Resuelve numricamente el ejercicio 2.a usando 
    un mtodo determinado.
    """
    x0 = 0.0
    y0 = 0.0
    xf = 0.4  # Para evitar la asntota en x = 0.408248...
    n  = nd(x0, xf, 0.1)  # Nmero de divisiones segn dx
    x, y = metodo(f2a, x0, y0, xf, n)
    error = [round(error_porcentual(xi, yi, sol2a), 2)
             for xi, yi in zip(x, y)]
    return x, y, error

resultados_2a_euler = resolver_2a(euler)
tabla_2a_euler = generar_tabla(sol2a, *resultados_2a_euler)
tabla_2a_euler

2.b.¶

Tenemos que resolver el siguiente problema de valor inicial:

$$ \left\{ \begin{align*} y' &= e^{x+y}\\ y(0) &= 0 \end{align*} \right. $$

Primero resolvemos la ecuacin diferencial de forma analtica por separacin de variables:

$$ \begin{align*} y' &= e^{x+y} \\ \frac{dy}{dx} &= e^{x+y} \\ \frac{dy}{dx} &= e^{x}\,e^{y} \\ \frac{dy}{e^{y}} &= e^{x}\,dx \\ \int\!{e^{-y}}\,dy &= \int\!e^{x}\,dx \end{align*} $$

Integrando a ambos lados tenemos la solucin general:

$$ \begin{align*} \frac{e^{-y}}{-1} &= {e^{x}} + C \\ -e^{-y} &= {e^{x}} + C \\ \end{align*} $$

Calculamos la constante de integracin dada la condicin inicial $y(0) = 0$:

$$ -e^{-0} = e^{0} + C \\ -1 = 1 + C \\ \Rightarrow C = -2 $$

Luego reemplazamos esta constante en la solucin general y despejamos la $y$ en trminos de $x$ para tener la solucin particular de la ecuacin diferencial:

$$ \begin{align*} -e^{-y} &= {e^{x}} - 2 \\ e^{-y} &= -{e^{x}} + 2 \\ ln(e^{-y}) &= ln(2 -{e^{x}})\\ {-y} &= ln(2 -{e^{x}}) \\ {y} &= -ln(2 -{e^{x}}) \\ \end{align*} $$

Antes de continuar, hay que tomar en cuenta que la funcin logaritmo neperiano solo est definida para valores en los que su argumento es mayor a cero:

$$ 2 -{e^{x}} > 0 \Rightarrow {e^{x}} < 2 \Rightarrow x < ln(2) \Rightarrow x < 0,6931471805599453 \text{ (aprox)} $$

Entonces, la funcin solucin est definida para valores menores a $ln(2)$, por lo que el intervalo no puede pasar dicho valor. Con esta informacin, procedemos a obtener la solucin numrica por el mtodo de Euler y su error porcentual en cada punto. Nos piden que utilicemos un valor de $\Delta x = 0.1$, por lo que hay que ajustar el nmero de divisiones (n) de acuerdo a esto.
