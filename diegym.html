Diegym

Training routines

For men: modified r/Fitness Basic Beginner Routine

Original routine.

Workout A

  • Barbell bent-over row: 3 sets, 5 reps, last set AMRAP.
  • Push-up: 3 sets, 5-8 reps, last set AMRAP.
    • Original: Bench press
  • Squat: 3 sets, 5 reps, last set AMRAP
    • Alternative: Front squat
    • Alternative: Hack squat
    • Alternative: Zercher squat

Workout B

  • Barbell bent-over row: 3 sets, 5 reps, last set AMRAP.
    • Original: Pull-up
  • Barbell military press: 3 sets, 5 reps, last set AMRAP.
  • Deadlift: 3 sets, 5 reps, last set AMRAP.

For women: modified Strong Curves Weeks 1-4 routine

Original routine.

Workout A

  • Glute bridge: 3 sets, 10-20 reps.
  • Dumbbell bent-over row: 3 sets, 8-12 reps (each side).
  • Weighted front squat: 3 sets, 10-20 reps.
  • Incline push-up: 3 sets, 8-12 reps.
  • Romanian deadlift: 3 sets, 10-20 reps.
  • Side lying hip abduction: 1 set, 15-30 reps (each side).
  • Front plank: 1 set, 20-120 seconds.
  • Side plank: 1 set, 20-60 seconds (each side).

Workout B

  • Single leg hip bridge: 3 sets, 10-20 reps (each side).
  • Dumbbell bent-over row: 3 sets, 8-12 reps (each side).
  • Dumbbell step-up: 3 sets, 10-20 reps (each side).
  • Dumbbell shoulder press: 3 sets, 8-12 reps.
  • Bird dog: 3 sets, 10-20 reps.
  • Side lying hip abduction: 1 set, 15-30 reps (each side).
  • Front plank: 1 set, 20-120 seconds.
  • Side plank: 1 set, 20-60 seconds (each side).
