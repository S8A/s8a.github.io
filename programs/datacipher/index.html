DataCipher

Last update: Jul 2017

Tool for encrypting and decrypting text using various classical ciphers. Made in Python.

  • ASCII: binary, octal, decimal, hexadecimal.
  • Atbash cipher.
  • Caesar cipher.
  • Vigenère cipher.
  • Columnar transposition (bad implementation).
